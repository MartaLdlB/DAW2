Elementos del DOM

Son los nodos del árbol DOM y existen 12 tipos diferentes. Los más relevantes son:

  • Document, nodo raíz del que derivan todos los demás nodos del árbol.
  • Element, representa cada una de las etiquetas XHTML. Se trata del único nodo que puede contener atributos y el único del que pueden derivar otros nodos.
  • Attr, se define un nodo de este tipo para representar cada uno de los atributos de las etiquetas XHTML, es decir, uno por cada par atributo:valor.
  • Text, nodo que contiene el texto encerrado por una etiqueta XHTML.
  • Comment, representa los comentarios incluidos en la página XHTML.

Crear y eliminar nodos

Para crear un elemento, se siguen cuatro pasos:

  1. Creación de un nodo Element que represente al elemento.
    Usa createElement(etiqueta) para ello.
  2. Creación de un nodo Text que represente el contenido del elemento.
    Usa createTextNode(contenido) para ello.
  3. Añadir el nodo Text como nodo hijo del nodo Element.
    Usa nodoPadre.appendChild(nodoHijo) para ello.
  4. Añadir el nodo Element a la página, en forma de nodo hijo del nodo correspondiente al lugar en el que se quiere insertar el elemento.
    Vuelve a usar nodoPadre.appendChild(nodoHijo) para ello.

Para eliminar un nodo del DOM bastará con utilizar la función removeChild().

Para insertar un nodo en una posición concreta, se debe indicar la posición deseada. Usaremos:

  • insertBefore(nuevo, posterior si el nodo padre de nuestro nuevo elemento tiene otros nodos hijo
  • appendChild() si el nodo padre no tiene más hijos.
  • replaceChild(nuevo, antiguo) si el padre tiene otros nodos hijo y queremos reemplazar uno de ellos por el nuevo. El nodo retirado desaparece del árbol DOM, pero sigue existiendo y puede referenciarse.
