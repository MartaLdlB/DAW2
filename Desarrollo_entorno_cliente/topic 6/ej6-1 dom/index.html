DOM

Es la estructura del documento HTML (Document Object Model) que puede leerse en forma de árbol, cuyos nodos son los elementos que la forman.


Document

Es un objeto que representa el árbol DOM de una página. Para acceder al DOM con JavaScript se hace a través de este objeto.


Acceso a elementos

Javascript nos proporciona un conjunto de herramientas para trabajar de forma nativa con el DOM. Podremos: buscar, crear, insertar/eliminar, modificar atributos y navegar por el documento.
