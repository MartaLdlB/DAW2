Otras funcionalidades de Window

find
find(text): indica si existe un texto en la página
fullScreen
indica si la aplicación se muestra a pantalla completa
location
objeto con información sobre la URL del sitio. Se pude usar para redirigir el sitio, modificando su valor.
open
open(textoURL): abre una URL en una nueva pestaña.
scroll*
acciones con scroll
  • scroll(x, y): desplaza a la posición indicada. También se le puede indicar un objeto tipo scroll, que contiene las propiedades top, left y behavior. Es equivalente a scrollTo().
  • scrollBy(x, y): desplaza los píxeles indicados
  • scrollX / scrollY: devuelve los píxeles desplazados en horizontal / vertical
setTimeOut
setTimeOut(action, time): espera los milisegundos indicados y, después, realiza una acción
